# 上海市初中学习系统 - Streamlit前端

## 📋 项目说明

这是上海市初中学习系统的Streamlit前端应用，适配华为PAD和iPad设备。

### ✨ 核心功能

- ✅ 用户登录/注册（Mock模拟）
- ✅ 首页学科选择
- ✅ 课程中心（课程列表）
- ✅ 课程详情查看
- ✅ 智能练习（题目展示）
- ✅ 个人中心（**API Key配置**）
- ✅ PAD端响应式适配

## 🚀 快速开始

### 1. 安装依赖

```bash
cd 前端
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

应用将在 `http://localhost:8501` 启动

### 3. 登录测试

**开发模式（Mock数据）：**
- 用户名：任意
- 密码：任意
- 任何用户名和密码都可以登录

**推荐测试账号：**
- 用户名：admin
- 密码：123456

## 🎯 功能说明

### 1. 用户认证（Mock模拟）

- **登录**：任意用户名密码可登录
- **注册**：填写表单后模拟注册成功
- **状态管理**：使用Session State管理登录状态

### 2. 课程中心

- **学科选择**：语文、数学、英语
- **年级选择**：初一、初二、初三
- **课程列表**：显示Mock课程数据
- **进度显示**：课程学习进度

### 3. 课程详情

- **课程信息**：标题、描述、关键词
- **知识点总结**：查看/生成按钮（待实现）
- **智能练习**：跳转到练习页面
- **学习进度**：更新学习进度（待实现）

### 4. 智能练习

- **题目生成**：Mock生成3道示例题
- **题型**：选择题、填空题、简答题
- **答题**：支持答题输入
- **导航**：题目导航和进度显示
- **提交**：批改功能（待实现）

### 5. 个人中心 ⭐**核心功能**

- **基本信息**：用户名、邮箱、学校、年级
- **AI API Key配置**：
  - 选择AI模型（DeepSeek/GPT-4）
  - 输入API Key
  - 保存配置（待连接后端）
  - 测试连接（待实现）
- **学习统计**：Mock统计数据
- **账号管理**：修改密码、退出登录
